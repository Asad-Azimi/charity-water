# Water Drop Dash

A simple charity: water themed browser game built with HTML, CSS, and JavaScript.

## How to play

- Press **Start Game**.
- Use the left and right arrow keys, or the on-screen buttons, to move the bucket.
- Catch clean water drops to gain points.
- Avoid dirty drops because they subtract points.
- Collect 20 clean drops before the 30-second timer ends.
- Press **Reset** any time to restart the game state.

## Features included

- Score tracking
- 30-second timer
- Dirty drop obstacle that reduces the score
- Reset button
- Random win/lose messages
- Confetti celebration when the player wins
- Responsive layout for smaller screens
- charity: water inspired colors

## Files

- `index.html` - page structure
- `style.css` - layout, branding, responsiveness, and animations
- `script.js` - game logic, timer, scoring, movement, reset, and confetti
